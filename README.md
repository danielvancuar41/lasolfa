# Gestionale Acquisti — Osteria La Sol Fa

Gestionale per la tracciatura degli acquisti dal fornitore Conad.

## Come usare

Password di accesso: `1234`

## Deploy su Vercel

1. Carica questa cartella su GitHub
2. Vai su [vercel.com](https://vercel.com) → New Project
3. Importa il repo GitHub
4. Clicca **Deploy** — nessuna configurazione necessaria

## Funzionalità

- Acquisti: aggiungi prodotti con quantità, prezzo e data
- Export Excel (CSV) scaricabile in qualsiasi momento
- Catalogo prodotti con immagini e ricette collegate
- Analisi con grafici per categoria, trend, top prodotti
- Archivio storico con filtri per data e categoria
- Tutti i dati salvati nel browser (localStorage)
